    ONLYOFFICE部署

    # 官网

    • 选择官网安装方式 点击查看 (opens new window)或者跟着教程一步步来：

    # Windows 下 Docker 安装 ONLYOFFICE

    docker run -i -t -d -p 6257:80 --restart=always -e "JWT_SECRET=ywjdsb" -e "JWT_ENABLED=false" --name onlyoffice onlyoffice/documentserver
    • 如果没有安装Docker，请先安装Docker，如何安装？点击查看安装教程
    • 如果镜像内没有ONLYOFFICE，以上命令会自动拉取镜像，拉取完成后会自动启动onlyoffice服务。
    • 默认密码为ywjdsb，这是框架中文档的多人协作密码，不能改。输入以下命令，如果输出ywjdsb，则设置成功。
    • -e "JWT_ENABLED=false"是禁用ONLYOFFICE的token验证，问就是懒得弄。
    docker exec onlyoffice /var/www/onlyoffice/documentserver/npm/json -f /etc/onlyoffice/documentserver/local.json 'services.CoAuthoring.secret.session.string'
    • 启动成功后，访问 http://localhost:6257 (opens new window) 验证ONLYOFFICE是否正常运行。

    # ONLYOFFICE 提示下载失败

    • 需要编辑配置文件允许私有ip通过
    # 进入容器
    docker exec -it onlyoffice bash
    # 编辑default.json配置文件
    nano /etc/onlyoffice/documentserver/default.json
    # 修改配置
    • 找到配置文件中的 request-filtering-agent 部分，修改 allowPrivateIPAddress 和 allowMetaIPAddress 为 true。例如：
    "request-filtering-agent" : {
  "allowPrivateIPAddress": true,
  "allowMetaIPAddress": true
},
    • 修改完成后按 Ctrl+X 保存并退出，提示输入 Y 保存，接着按 Enter 退出编辑器。
    • 输入exit退出容器。
    # 重启容器
    docker restart onlyoffice
