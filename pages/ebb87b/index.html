如何安装

# Windows下安装

前往官网 (opens new window)下载docker.desktop即可

  • 如果无法启动，请检查电脑是否已开启虚拟化。
  • 控制面板 -> 程序 -> 启用或关闭Windows功能 -> 勾选Hyper-V

# 设置 Docker Engine

  • 拉取镜像速度慢？点击右上角Settings按钮，选择Docker Engine，替换为以下配置：
{
  "builder": {
    "gc": {
      "defaultKeepStorage": "20GB",
      "enabled": true
    }
  },
  "experimental": false,
  "features": {
    "buildkit": true
  },
  "registry-mirrors": [
    "https://registry.docker-cn.com",
    "http://hub-mirror.c.163.com",
    "https://docker.mirrors.ustc.edu.cn",
    "https://cr.console.aliyun.com",
    "https://mirror.ccs.tencentyun.com"
  ]
}

# Linux下安装

# 1. 查看是 CentOS 还是 Ubuntu

cat /etc/issue

# 2. 安装依赖

    sudo yum install -y yum-utils

    sudo apt update
sudo apt install -y ca-certificates curl gnupg lsb-release

    // Make sure to add code blocks to your code group

    # 3. 如果需要，卸载旧版本

      sudo yum remove docker docker-client docker-client-latest docker-common docker-latest docker-latest-logrotate docker-logrotate docker-engine

      sudo apt remove docker docker-engine docker.io containerd runc

      // Make sure to add code blocks to your code group

      # 4. 添加官方仓库

        sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

        # 1.添加 Docker 的 GPG 密钥：
sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

# 2.设置 Docker 仓库:
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

        // Make sure to add code blocks to your code group

        # 5. 安装 Docker

          sudo yum install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

          sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

          // Make sure to add code blocks to your code group

          # 6. 启动并设置开机自启

          sudo systemctl start docker
sudo systemctl enable docker

          # 7. 验证Docker是否安装成功

          docker --version

          # 8. 安装docker-compose

          # 下载最新版本的docker-compose
sudo curl -L "https://github.com/docker/compose/releases/download/$(curl -s https://api.github.com/repos/docker/compose/releases/latest | jq -r .tag_name)/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

          # 9. 给docker-compose添加可执行权限

          sudo chmod +x /usr/local/bin/docker-compose

          # 10. 验证docker-compose是否安装成功

          docker-compose --version
